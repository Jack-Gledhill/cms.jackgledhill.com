# crud.jackgledhill.com

An extensible CRUD database API for managing the content on jackgledhill.com.

## Project Aims & Purpose

This project aims to be a simple, extensible CRUD API written in Go.
To that end, the project will follow the Repository Design Pattern, to create a decoupled and highly testable architecture.

The project serves as more of a learning exercise as opposed to a solution to an actual problem.
